Technologies Used

Python

Primary programming language for the project. Chosen for its simplicity and powerful libraries.

Tor Network

Enables access to Dark Web resources while maintaining anonymity.

Requests Library

Python library for making HTTP requests. Simplifies communication with Onion sites.

BeautifulSoup

Library for parsing HTML and XML documents. Used for content extraction.

Tkinter

Python's standard GUI toolkit. Used for creating the result viewer interface.
