Project Objectives

Key goals of the Onion Web Crawler & Risk Analysis System:

Web Crawling

Automatically discover and traverse Onion sites through the Tor network

Content Extraction

Isolate and retrieve text content from crawled web pages

Keyword Detection

Identify suspicious or prohibited keywords in extracted content

Classification

Categorize content based on site type and keyword analysis

Risk Assignment

Assign risk levels based on classification and keyword severity

GUI Display

Present results in a color-coded interface for easy interpretation
